
# API Team 2

DeepCytes Forensics Task 



## Run Locally

Clone the project

```bash
    git clone https://github.com/Baboon12/API-Team-2.git
```

Navigate to the directory
```bash
    cd API-Team-2
```

Install Dependencies
```bash
    pip install Flask pymongo
```

Run 
```bash
    python app.py
```
